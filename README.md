# ComicVineScraper

Display popular volumes and issues from Comic Vine.

Patterns:

    No storyboards
    UIPageViewController
    UISegmentedControl
    MVVM/Coordinator (based on TMDB by Maksym)
    PageViewControllerSegmentedAdapter (from dchohfi)

Frameworks:

    Combine
    Kanna to parse HTML

Feel free to use.
